================================================================================
AUTHORIZATION LEVELS
================================================================================

1. Public - Không cần xác thực
2. Authenticated - Cần đăng nhập
3. ADMIN - Chỉ người dùng có quyền ADMIN


================================================================================
NOTES / GHI CHÚ
================================================================================

- Các endpoints được tổ chức thành 6 module chính
- Hệ thống sử dụng JWT Token cho xác thực
- Hầu hết các endpoints return Response object
- Sản phẩm hỗ trợ upload hình ảnh
- Giao dịch hỗ trợ 3 loại: purchase (mua), sell (bán), return (hoàn trả)
